## 1. 판정 코드

| 코드 | 명칭 | 의미 | 종합 집계 |
|------|------|------|-----------|
| **P** | 적정 | 기준 충족 | 적정 카운트 |
| **W** | 주의·확인필요 | 추가 확인·자료 보완 필요 | 주의 카운트 |
| **F** | 부적정 | 기준 위반·즉시 조치 | **부적정 1개 이상 → 종합 F** |
| **N/A** | 해당 없음 | 비목·자료 조건상 점검 대상 아님 | **집계에서 제외** |
| **U** | 미판정 | 자료 없어 자동 판정 불가 (선택 코드) | W와 동일 취급 가능 |

> 기존 SAFE: P / W / F 만 사용. **N/A** 는 2단계부터 UI·DB에 신설.

---

## 2. N/A vs W(확인필요) — 구분 기준 (작업안)

| 상황 | 권장 판정 | 이유 |
|------|-----------|------|
| 계획에 **인건비 비목 없음** → 원천징수 항목 | **N/A** | 점검 자체가 성립하지 않음 |
| **③ 지출증빙 없음** → 증빙 완전성·원천징수 등 ③ 전용 | **N/A** | 해당 사업 유형에서 증빙 제출 의무 없음 |
| **③ 없음** → 집행↔증빙 **교차** 항목 (X03, X04) | **W (확인필요)** | 검증은 필요하나 자료로 확인 불가 |
| ① 미제출 → 예산·비목 기준 항목 | **W** | 프로필 수동 입력 전까지 판정 보류 |
| 유사도만 낮고 규칙·교차 근거 없음 | **W** | **F로 단독 확정 금지** (정성 항목) |

### V3(③ 없음) 확정안 (팀 확인 필요)

| 항목 유형 | 기대 판정 |
|-----------|-----------|
| ③ 전용 (증빙 완전성, 세금계산서 등) | **N/A** |
| ②+③ 교차 (금액·건수 대조) | **W** — 「증빙 미제출로 대조 불가」 |
| ②만으로 판단 가능 (JC-02 비목 적정 등) | **적용** (P/W/F) |
| ①+② 교차 (X07 사업기간 내 집행) | **적용** — ① 기간 vs ② 집행일. ① 없으면 **W** |

## 3. 종합 판정 규칙 (작업안)

```
종합 판정:
  IF 적용 항목 중 F ≥ 1  → 종합 F
  ELIF 적용 항목 중 W ≥ 1 → 종합 W
  ELSE                    → 종합 P

적용 항목 = 전체 − N/A
```

| 질문 | 작업안 |
|------|--------|
| F 1개면 전체 F? | **예** (현행 SAFE와 동일) |
| N/A는 종합에서? | **제외** (분모에서 빼고 집계) |
| W만 있으면? | 종합 **W** (적정 아님) |

## 4. 판정 방식별 기본값

| `check_method` | 자동 F 가능 | 비고 |
|----------------|------------|------|
| `rule` (규칙검증) | 예 | 수치·필드 조건 |
| `cross` (교차) | 예 | 한쪽 자료 없으면 W |
| `similarity` (유사도) | **원칙적 W** | 임계값 미만 → W, F는 규칙 보조 시만 |
| `manual` | — | 담당자 확정 |

## 6. 실무 일제점검 ↔ SAFE 판정 대응 (2026-07-01 확정안)

> **출처:** `지방보조금 부정수급 일제점검 실시계획_체크리스트.pdf`  
> **상세:** `08_일제점검_체크리스트_항목.xlsx` 시트「판정매핑」

| 실무 (일제점검) | SAFE 자동 | 최종 보고 (3값) | 비고 |
|-----------------|-----------|-----------------|------|
| **부정/과오** | **F** (부적정) | 부정과오 **O** | |
| **부정아님** | **P** (적정) | 부정과오 **X** | |
| **해당없음** | **N/A** | **해당없음** | ② 집행건·구비 대상 없음 |
| **판단보류** | **W** (확인필요) | *(자동만, 최종 제외)* | 담당자 확정 시 O/X/N/A |

**원칙:** SAFE 엔진은 **4값(P/F/W/N/A)**. 최종 보고·골든케이스 기대값은 **3값(O/X/해당없음)** 가능.  
**판단보류**는 삭제하지 않고 **W**로 유지 → 담당자가 소명·확인 후 O/X/N/A로 확정.

### 해당없음 (실무 예시 — 일제점검과 동일)

| 항목 | 조건 | 판정 |
|------|------|------|
| J101-02 원천징수·급여대장 | ② **101 집행건 없음** | **N/A** |
| J20133-* 임차 항목 | ② **201-33 집행 없음** | **N/A** |
| J20101-02 증빙 | ③ **미제출** | **N/A** |
| JC-01 계획vs집행 | ①②만 있으면 적용 (③ 무관) | **규칙(P/W/F)** |
