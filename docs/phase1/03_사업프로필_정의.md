# 03. 사업 프로필 정의 (비목 중심)

> **전제:** `01_사업검토_정의서.md`  
> **비목 출처:** 가이드 붙임1 p.48~58 (`붙임1_2025년 지방보조금 관리 가이드.pdf`)

---

## 1. 프로필이란

**「이 사업에 어떤 점검 항목을 적용할지」** 를 정하는 메타데이터.

- 사업 **종류·명칭**으로 판단하지 **않음**.
- **사업계획서 예산집행계획**에 어떤 **보조비목(편성목)·세목**이 있는지가 핵심.

---

## 2. 프로필 구성 (3블록)

### 블록 A — 식별 (판정에 미사용)

| 필드 | 설명 |
|------|------|
| 사업명 | 제목·식별용 only |
| 사업연도 | |
| 수행기관 | 선택 |

### 블록 B — 예산집행계획 비목 (핵심)

가이드 붙임1 체계에 따른 **편성 여부 (Y/N)** 또는 **계획 금액**.

**Phase 1 최소 세트 (대분류)** — 상세 목록은 `03B_편성목_대분류목록.md` 참고

| 코드 | 편성목 | 점검 연계 예 |
|------|--------|-------------|
| B101 | 인건비 | 원천징수, 4대보험, 인건비 비율 |
| B201 | 일반운영비 | 용역(201-34), 행사(201-03), 급량(201-01 내), 운영수당(201-32) |
| B202 | 여비 | 출장·여비 증빙 (202-01/03) |
| B203 | 업무추진비 | 간담회·카드 사용 (203-01~04) |
| B401 | 시설비 및 부대비 | 공사·설계·감리 (401-01~03) — **용역은 B201** |
| B405 | 자산·물품취득비 | 물품 구매 (405-01) |
| … | (가이드 p.48~58 참고) | |

**규칙**

- `B101 = N` → 인건비 관련 체크리스트·규칙검증 **N/A**
- `B101 = Y` 이고 ②에 집행 없음 → 「미집행·확인필요」 등 별도 정책

### 블록 C — 자료 존재

| 필드 | 값 |
|------|-----|
| has_plan | ① 유무 |
| has_execution | ② 유무 |
| has_proof | ③ 유무 (**없어도 검토 가능**) |
| has_settlement | ④ 유무 |

→ 자료별 항목 + 교차 항목의 **적용/스킵**에 사용.

---

## 3. 항목 적용 로직 (2·3단계 구현 목표)

```
IF 항목.required_docs ⊄ 제출된 자료 → N/A 또는 확인필요
IF 항목.required_budget_item ⊄ 프로필(예산집행계획) → N/A
IF 프로필에 비목 있음 AND 집행·증빙 있음 → 해당 체크리스트·규칙 적용
```

**사업종류는 이 식에 넣지 않음.**
