## N/A 판정 규칙 (일제점검 ↔ SAFE)

```
② 집행내역에 해당 세목(101, 201-33, 401-01 등) 집행건 없음
  → 해당 세목 점검항목 전부 N/A (해당없음)

③ 미제출
  → ③ 전용 항목: N/A
  → ②+③ 교차(JC-03 등): W  (JC-01은 ①②만 → ③ 없어도 적용)
  → ②만 가능(JC-02 비목 적정): 적용
  → ①+② 교차(X07 사업기간 내 집행): 적용 (①·② 있을 때)
```

### G6 JC-02 / X07 분리 (팀 합의, 2026-07-01)

| 항목ID | 점검 내용 | 구비 | SAFE기대 | 비고 |
|--------|-----------|------|----------|------|
| **JC-02** | 집행 건별 내용이 **비목에 적정**한지 | ② | 적용 | 일제점검 JC-02 원 정의 |
| **X07** | **사업기간 내** 집행 여부 | ①,② | 적용 | ①에 적힌 기간 vs ② 집행일 대조. **기간 자체가 옳은지는 검증하지 않음** |
| *(① 없음 시 X07)* | — | — | **W** | 사업기간 기준 없음 |

> 구버전 골든케이스의 「JC-02 = 집행기간 적정」 표기는 **오류**였으며, 사업기간 검증은 **X07**으로 분리함.
