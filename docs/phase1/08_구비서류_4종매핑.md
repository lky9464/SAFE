# 08-B. 구비서류 — SAFE 4종 자료 매핑

> **출처:** 일제점검 체크리스트 p.1 (주체·준비자료)  
> **원칙:** 4종 외 서류는 **주로 ③ 지출증빙 ZIP**에 포함. 항목별 `required_evidence`로 구분.

---

## 1. SAFE 4종 ↔ 구비서류

| 체크리스트 구비서류 | SAFE | 포함 내용 예 | 비고 |
|---------------------|------|-------------|------|
| 지방보조사업 관련 제출 서류 | **①** | 사업기본정보, 수행기관, 세부추진계획, 재원조달, **예산집행계획**, 공모결과, 수행계획, 변경신청 | ① 핵심 |
| 참여인력 상세 서류 | **②+③** | 참여인력현황표, 건강보험 가입내역 | ② 집행 + ③ 증빙 |
| 업무수행기록 | **③** (일부 ②) | 기안문, 보고, 업무일지, 회의, 출장 | ZIP 내 품의서 |
| 가족관계 참여자 증빙 | **③** | 업무수행기록, 건강보험자격득실 | 인건비 항목 연계 |
| 사업 결과물(실물·사진) | **③** 또는 별도 | 산출물 증빙 | 사업별 상이 |
| 실적보고서 | **④** | 집행금액 전체 내역 | |
| 회계·감사 보고서 | **④** 또는 **③** | 10억 이상 법 제18조 감사 | |
| 근태관리 자료 | **③** | 출근부, 출입기록, 외부수행 내역 | 인건비·여비 연계 |
| 거래처 비교견적·사유서 | **③** | 거래당 견적서, 전자세금계산서 미발행 사유서 | |
